
# 현재 문제점
- 스팀이나 인디 고고 같은 퍼블리셔에 올리지 못하는 작은 소규모 인디게임 회사들은 게임 혹은 굿즈 판매활로 를 찾기 힘듬 
- 유저들에게 선보일 기회들도 찾기 힘듬

# 해결 방안
- 코인을 발행해서 유저와 게임사를 직접 연동을 시킴
- 인디게임 퍼블리셔를 해주고 유저가 직접 게임의 펀딩을 할수있는 기회를 제공
